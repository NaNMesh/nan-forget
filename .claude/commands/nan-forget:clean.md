# nan-forget:clean — Garbage Collection

Run garbage collection on stale memories. Archives decayed, expired, and duplicate entries. Syncs MEMORY.md.

## Instructions

Try `memory_clean` MCP tool first. If unavailable, run `npx nan-forget clean` via Bash.

Display the results: archived (decayed), archived (expired), archived (deduped), MEMORY.md synced, duration.

# nan-forget:compact — Consolidate Memories

Force consolidation of aging memories into compact long-term entries. Clusters related memories by topic, summarizes them, and archives the originals.

## Instructions

Try `memory_consolidate` MCP tool first. If unavailable, run `npx nan-forget consolidate` via Bash.

Optionally accepts a project name: `/nan-forget:compact my-project`

If `$ARGUMENTS` is provided, pass it as the project filter.

Display the results: clusters found, memories consolidated, new entries created, duration.
